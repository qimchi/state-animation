# Transaction Details Image

Drop your transaction details page screenshot/image here.

## Instructions

1. Place your transaction details page image in this folder
2. Name it: `tx-details.png` (or `.jpg`, `.jpeg`, `.svg`)
3. The image will automatically be used in the success animation reveal

## Supported Formats
- PNG (`.png`)
- JPEG (`.jpg`, `.jpeg`)
- SVG (`.svg`)

## Image Requirements
- Should be a screenshot of the transaction details page
- Will be displayed at full width within the mobile frame
- Recommended: Match the mobile frame dimensions (280x600px or similar aspect ratio)
